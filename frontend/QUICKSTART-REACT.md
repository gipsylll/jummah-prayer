# Быстрый старт React версии

## Установка зависимостей

```bash
cd frontend
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

**Важно:** Убедитесь, что C++ бэкенд запущен на порту 8080!

## Что было сделано

✅ Полная миграция на React 18
✅ React Router для навигации
✅ Context API для управления состоянием
✅ Все основные страницы перенесены
✅ Авторизация работает
✅ Темная тема работает
✅ Многоязычность работает
✅ Интеграция с существующим C++ бэкендом

## Структура

- `src/components/` - Переиспользуемые компоненты
- `src/pages/` - Страницы приложения
- `src/contexts/` - React контексты (Theme, Auth, PrayerTimes)
- `src/services/` - Сервисы для работы с API
- `src/utils/` - Утилиты (переводы и т.д.)
- `src/styles/` - Стили (скопированы из оригинального проекта)

## Отличия от оригинальной версии

- Используется React Router вместо ручной навигации
- Состояние управляется через Context API
- Компонентный подход вместо прямого DOM манипулирования
- Более структурированный код

Старая версия (vanilla JS) сохранена в файлах `js/`, `index-old.html` (если нужно).
